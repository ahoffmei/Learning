# H3 Indexing 
Data structure designed to improve the performance of spacial queries. 
Does this by breaking up geographical regions into hexagonal data segments. 

See the following website for a visual example: https://h3geo.org/

Esentially, we are creating a heat map that we can use for breaking up data segments.
