# TCP/IP Stack 
Has 4 layers:
- Application Layer
- Transport Layer
- Network Layer (IP)
- Link Layer 

# Application Layer 
- HTTP
- SMTP (email protocol)
- FTP (File Transfer Protocols)


# Transport Layer 
Ensures reliable data transfer between applications. 
TCP: Reliable data transmission 
UDP: Faster but less reliable 

# Network Layer
IPv4 and IPv6, frameworks for communication.
